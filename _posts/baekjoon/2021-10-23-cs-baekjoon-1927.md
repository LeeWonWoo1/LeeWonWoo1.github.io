## 2. 핵심 아이디어

- 최소 힙의 기본적인 기능을 구현
- Python에서 heapq 라이브러리를 이용하면 간단히 힙 구현 가능

<br>

## 3. Python 문제풀이

```python
# 코드 1
import heapq
import sys
input = sys.stdin.readline

n = int(input())
heap = []
result = []

for _ in range(n) :
  data = int(input())
  if data == 0 :
    if heap :
      result.append(heapq.heappop(heap))
    else :
      result.append(0)
  else :
    heapq.heappush(heap, data)

for data in result :
  print(data)

# 코드 2
import sys
import heapq
input = sys.stdin.readline

n = int(input())
arr = []

for _ in range(n) :
  x = int(input())
  if x == 0 :
    try :
      print(heapq.heappop(arr))
    except :
      print(0)
  else :
    heapq.heappush(arr, x)
```
